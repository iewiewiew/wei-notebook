## Ubuntu16

### 1. 拉取 Ubuntu 16.04 官方镜像

镜像地址：https://docker.aityp.com/image/docker.io/ubuntu:16.04?platform=linux/386

docker pull swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/ubuntu:16.04-linux386

### 2. 启动容器（以特权模式，暴露 22 端口，命名为 ubuntu16-ssh）
docker run -itd --name ubuntu16-ssh -p 3333:22 --privileged swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/ubuntu:16.04-linux386 /bin/bash

### 3. 进入容器

docker exec -it ubuntu16-ssh /bin/bash

### 4. 在容器内执行以下命令（安装并配置 SSH）

更新源（可选，加速安装）  
apt update

安装 openssh-server（默认就是 7.2p2 版本）  
apt install -y openssh-server

创建 SSH 运行目录  
mkdir -p /var/run/sshd

允许 root 登录 SSH（修改配置）  
sed -i 's/^PermitRootLogin prohibit-password/PermitRootLogin yes/' /etc/ssh/sshd_config

设置 root 密码（这里设为 123456，可自定义）  
echo 'root:123456' | chpasswd  

启动 SSH 服务  
/usr/sbin/sshd -D &

### 5. 安装 vim git

apt install -y vim  
apt install -y git  

ssh-keygen
cat /root/.ssh/id_rsa.pub

### 6. 验证 OpenSSH 版本

ssh -V 2>&1  # 输出：OpenSSH_7.2p2 Ubuntu-4ubuntu2.10, OpenSSL 1.0.2g  1 Mar 2016

### 7. 清理数据

docker ps -a |grep ubuntu16-ssh

docker rm -f ubuntu16-ssh

docker images |grep ubuntu

docker rmi -f swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/ubuntu:16.04-linux386



## Debian13

### 1. 拉取 Debian 13 官方镜像

镜像地址：https://docker.aityp.com/image/docker.io/debian:13

docker pull swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/debian:13

### 2. 启动容器（特权模式，暴露 4444 端口）

docker run -itd --name debian13-ssh -p 4444:22 --privileged swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/debian:13 /bin/bash

### 3. 进入容器
docker exec -it debian13-ssh /bin/bash

### 4. 容器内安装并配置 SSH

```
apt update && apt install -y openssh-server
mkdir -p /var/run/sshd
sed -i 's/^PermitRootLogin prohibit-password/PermitRootLogin yes/' /etc/ssh/sshd_config
echo 'root:123456' | chpasswd
/usr/sbin/sshd -D &  # 后台启动 SSH 服务
```

### 5. 安装 vim git

```
apt install -y vim
apt install -y git
ssh-keygen
ssh-keygen -t rsa -b 2048 -C "你的备注（如邮箱）"
cat /root/.ssh/id_rsa.pub
```

### 6. 添加 bookworm-backports 源

vim /etc/apt/sources.list.d/bookworm-backports.sources

```
Types: deb deb-src
URIs: http://deb.debian.org/debian
Suites: trixie-backports
Components: main
Enabled: yes
Signed-By: /usr/share/keyrings/debian-archive-keyring.gpg
```

更新
```
apt-get update
apt-get install openssh-client/trixie-backports
```

### 7. 清理数据

docker ps -a |grep debian13-ssh  
docker rm -f debian13-ssh  

docker images |grep debian   
docker rmi -f swr.cn-north-4.myhuaweicloud.com/ddn-k8s/docker.io/library/debian:12
