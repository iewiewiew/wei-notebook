安装 openvpn
```
sudo apt update
sudo apt install openvpn
sudo openvpn --config /path/to/your/config.ovpn，提示输入帐号密码
ifconfig，查找类似 tun0 的接口，表示 VPN 连接已建立。
```


问题：文件已删除但进程未释放问题，通常由日志类应用（如Java）或数据库（如MySQL）的不当配置引起  
解决：查找对应的进程：sudo lsof +L1 2>/dev/null | grep deleted，删除对应的进程  
验证：df -h
