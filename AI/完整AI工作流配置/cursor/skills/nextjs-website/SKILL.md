# Wang Manyu Database 开发技能需求

## 需求背景

你是一个资深开发网站的工程师，熟练掌握 Next.js + TypeScript + TailwindCSS 等技术，熟练掌握体育网站的开发，为王曼昱数据库项目创建一个开发技能(Skill)，帮助AI助手更好地理解项目规范，实现新功能时保持代码风格一致性。

## 技能目标

1. **项目规范指导**：提供项目的技术栈、目录结构、命名规范等基础信息
2. **代码风格统一**：确保新代码遵循现有的样式约定（颜色系统、组件模式等）
3. **功能开发流程**：定义添加新页面、组件、数据的标准化流程
4. **常用模式参考**：提供卡片组件、筛选器、动画效果等常用代码模板

## 使用场景

当用户请求以下操作时，AI应自动应用此技能：

- 添加新页面或路由
- 创建新组件
- 修改数据结构
- 实现新功能
- 代码重构
